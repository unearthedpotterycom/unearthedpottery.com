This white stoneware bowl has been high-fired up to Cone 10 (2300 F). It is dishwasher, microwave, oven safe. It was made in spring 2006.
All

The sunburst bowl was a part of a set that was commissioned from a guest at Maho Bay Campground. Notice the variation in how the same glaze would fire in the same kiln load!
